# zakaat_counter-
A precise, user-friendly tool designed to simplify Zakat calculations. Built to help users accurately determine their contributions across various asset classes including cash, gold, silver, and investments.
A high-performance, aesthetically driven Zakat Calculator built with HTML, Tailwind CSS, and JavaScript. This tool features a "Dark Neon" UI with real-time logic to determine Zakat eligibility based on Shariah-compliant Nisab thresholds.

Key Technical Features:

Dynamic Asset Calculation: Handles Gold (Tola), Silver (Tola), Cash, and Receivables while automatically deducting Liabilities.

Dual-Nisab Logic: Implements smart conditional logic—checking against Gold Nisab (7.5 Tola) for gold-only assets and Silver Nisab for mixed assets.

Manual Market Rates: Users can manually override Gold and Silver prices to ensure accuracy based on their local market.

Glassmorphic UI: A premium dark-mode experience using Tailwind's backdrop-filter and radial gradients for a modern "FinTech" feel.
